MG Concrete Catalog

طريقة التشغيل:
1) فك الضغط عن الفولدر mg_catalog_web على Desktop.
2) افتح index.html في المتصفح.
3) لو عندك VS Code Live Server، افتح الفولدر واضغط Open with Live Server.
4) لتغيير رقم واتساب: افتح app.js وغير WHATSAPP_NUMBER.
5) لتغيير المنتجات: افتح products.json وعدل الاسم والسعر والصورة.
6) ضع صور المنتجات داخل فولدر images بنفس أسماء product1.jpg ... product5.jpg.
